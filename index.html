What is AllerGPS?


AllerGPS is a dynamic web application built for those who have allergies that
hinder them from eating certain foods.

It takes into account these allergies to chart a map of locations that the user can go to
and enjoy without worrying about their allergies.

AllerGPS was made to be simple, elegant, and powerful.

Features

AllerGPS uses AllerBase, a detailed database
of restaurant data that includes all of your favorite places and more to enable better searching!

AllerGPS has a fully-functioning map,
which means that anyone in the U.S.
can access it from anywhere!

Features

Unlike other similar services,
AllerGPS does not require any fee or sign up
for access to its search tool. It is available for free.

AllerGPS also has a variety of food allergies
for the user to choose in order to have
the best experience possible.

Come check out the source code for this website on our GitHub!
Feel free to clone the code and make a spinoff application!
